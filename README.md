java 版百度网盘功能，目前已经实现：
	1：百度网盘登录
	2：列出百度网盘文件
	3:切换目录
	4：多线程下载文件

待续...

	usage:
		登陆： 
			login 
			参数：
				-test 指定本地保存的用户名密码文件  默认在D:\BDLogin.txt
				-loc 读取已经登陆过的bduss 默认在D:\PcsLogin.txt
		列出当前文件夹：
			ls
				-f 指定文件路径（不加或者直接写路径也可以）
		切换目录：
			cd 
				-f 指定文件路径（不加或者直接写路径也可以）
		下载：
			download
				-f 指定文件路径（不加或者直接写路径也可以）
			
			注：文件名不能带空格，如果文件名的确有空格，加''(单引号)。
